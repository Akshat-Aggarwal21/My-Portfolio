## 🛠️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Akshat-Aggarwal21/My-Portfolio.git
```

Navigate to the project folder:

```bash
cd My-Porfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
